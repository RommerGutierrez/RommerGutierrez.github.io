Glosario de Términos

Elaborado con el objetivo de aclarar de manera breve y precisa el concepto de las palabras y términos técnicos abstractos presentes en el manual para facilitar la comprensión del mismo.

Letras

A

Actualización

Proceso de instalar la versión más reciente de un software o sistema operativo.

Administradores de sistemas

Personas responsables de gestionar y mantener los sistemas informáticos en buen funcionamiento.

Antivirus

Programa que detecta y elimina virus informáticos y otros tipos de malware.

Archivos temporales

Archivos creados por el sistema operativo o aplicaciones que son necesarios solo por un corto periodo de tiempo.

Arranque

Proceso de iniciar una computadora y cargar el sistema operativo.

C

Cables

Conductores que permiten la transmisión de datos o energía entre dispositivos

Componentes

Partes individuales que conforman una computadora, como la memoria RAM, el procesador, etc.

Computadora

Dispositivo electrónico que procesa datos y realiza tareas según las instrucciones de un programa.

Conexión a internet

Enlace que permite a una computadora acceder a la red global de información.

Configuraciones

Ajustes o parámetros que determinan el comportamiento de un sistema o software.

Contraseñas

Claves secretas usadas para acceder a sistemas o cuentas y mantener la seguridad.

Controladores

Programas que permiten que el sistema operativo se comunique con el hardware.

D

Desfragmentar

Proceso de reorganizar los datos en un disco duro para mejorar el rendimiento.

Diagnóstico

Evaluación para identificar problemas en el hardware o software.

Documentación

Registro detallado de los procedimientos y mantenimiento realizado.

E

Equipo

Término general para referirse a la computadora y sus componenteso.

H

Hardware

Componentes físicos de una computadora.

I

Inventario

Lista detallada de todos los equipos y componentes disponibles.

l

Limpieza

Proceso de eliminar polvo y otros contaminantes de los componentes físicos de la computadora.

M

Mantenimiento

Conjunto de acciones para asegurar que una computadora funcione correctamente.

O

Optimización

Mejorar el rendimiento de un sistema o software.

P

Procedimientos

Pasos detallados para realizar una tarea o resolver un problema.

R

Redes

Sistemas de comunicación que conectan múltiples dispositivos para compartir información.

Registro

Archivos del sistema operativo que contienen información sobre la configuración y operaciones del equipo.

Repuestos

Partes de reemplazo para reparar componentes dañados.

S

Seguridad informática

Prácticas y herramientas para proteger la información y sistemas de amenazas.

Sistema operativo

Software principal que gestiona el hardware y permite la ejecución de otros programas.

Software

Programas y aplicaciones que realizan tareas en una computadora.

Soluciones

Métodos o procedimientos aplicados para resolver problemas.

T

Técnicos

Personas especializadas en el mantenimiento y reparación de sistemas informáticos.
